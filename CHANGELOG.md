## [0.1.0] - 2025-11-08

### Added

- Initial release
- `HumanSize` for human-readable file sizes with `UnitSystem` (binary/decimal)
- `HumanDuration` for human-readable time durations with 'yesterday' and 'tomorrow' support
- `HumanNumber` for human-readable number formatting
- `HumanPercentage` for human-readable percentages
- Long forms of units
